# Digital Knowledge Network (DKN) - Frontend

## Prerequisites

Before running the frontend application, ensure you have the following installed:

### Required Software

- **Node.js** (v14.0 or higher recommended)
  
  Node.js is required to run the development server and build the application. You can download and install it from the official website:
  
  [Download Node.js](https://nodejs.org/en)


## Getting Started

Follow these steps to set up and run the frontend development server:

#### 1. Clone the Repository

```bash
git clone https://github.com/adhikarisanjib/dkn_mobile_web_assignment.git
```

#### 2. Navigate to Frontend Directory


```bash
cd frontend
```

#### 3. Install Dependencies

```bash
npm install
```

This will download and install all the packages listed in `package.json`.

#### 4. Start Development Server

```bash
npm run dev
```

The application will start and be accessible at the URL displayed in your terminal (typically `http://localhost:5173`).
